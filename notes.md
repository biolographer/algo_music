# MIDI setup with Logic Pro

- Don't use midi chan 0 for PC messages, it might be reserved for special purposes
- to send Program Change (PC) messages to Surge XT as a VST in Logic go to "Project Settings" > MIDI > General and check the box: "Send After Loading Project [] Used instrument midi settings"
    - found here: https://discussions.apple.com/thread/255478635?sortBy=rank
- then open surge xt as vst and select MIDI Program folder
